# Taquería Michoacán

Website for Taquería Michoacán.

## Technology Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- ESLint

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
